# FlyEasy-a-flight-management-app
FlyEasy is a flight booking management app that makes your travel experience seamless, fast, and hassle-free.
FlyEasy is a full-stack web-based Flight Management System built using **React.js**, **Node.js (Express)**, and **hardcoded flight data**. It allows users to search, book, cancel, or reschedule flights. Admins can manage flights, users, and bookings through a dedicated dashboard.

---

## 🚀 Features

### 👤 User Portal
- User registration & login with JWT
- Search flights by route, date, class
- Book, cancel, or reschedule tickets
- E-ticket display & booking history
- 20% discount on next booking (within 20 days)
- Responsive UI with Tailwind CSS

### 🛠️ Admin Panel
- Add/edit/delete flights
- View and manage users
- Monitor bookings & discount usage
- Export reports (CSV/PDF)

---

## 📁 Tech Stack

| Layer      | Technology                     |
|------------|--------------------------------|
| Frontend   | React.js, Redux Toolkit, Tailwind CSS |
| Backend    | Node.js, Express.js, JWT, Express-validator |
| State/API  | Axios for API calls, LocalStorage for token |
| Payment    | Mock integration (can use Razorpay/Stripe) |
| Data Store | Hardcoded JSON files (flights/bookings) or MongoDB (optional) |
